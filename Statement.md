## Problem statement
Many schools, hospitals in rural areas still use paper pen to keep track of the paitents. These practises are hard to search through and easy to lose. Along with that it's hard to spot which paitents requires more attention based on their medical condition

## Scope Of The Project
- VitalTrack solves this by digitilizing the records
  * It allows the user to save and input data permanently
  * It help identify high risk paitents.
  * It provides a visual summary
  * It is designed for a single user
## Target User 
* School Nurse
* Small Clinic staff
* Personal User

## High Levl Features
* Digital record keeping
* Safety alert
* Search Engine
* Visual Analytics
